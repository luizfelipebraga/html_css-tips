Let Me Ask

Projeto desenvolvido em ReactJs com TypeScript com integração no Firebase simulando uma espécie de banco de dados e BackEnd.
Plataforma similar ao Brainly, onde o usuário pode fazer perguntas e outras pessoas podem resolvê-las.
Portanto, uma aplicação básica para fins de como funciona o context e autenticação com o google.

Senai Vagas

Projeto desenvolvido em SQL, C# e ReactJs com TypeScript.
Plataforma similar ao linkedin em que apenas alunos SENAI poderiam se inscrever e empresas poderiam recrutá-los. Portanto, agilizando o processo de seleção e contratação.

Spring Bank Project

Mini Projeto em Java usando Spring Boot Framework afim de aprender e para aplicação de uma vaga na Orange, contendo validações e usando boas práticas como DTO's e error exception.
